🎤 HOW TO USE VOICE ASSISTANT

1. Click "🎙️ Voice Assistant" button in header
2. Select your language (English, Hindi, Tamil, Telugu, Bengali)
3. Click "Start Voice Command"
4. Allow microphone access when prompted
5. Speak clearly:
   • "Show my coverage"
   • "What's my premium?"
   • "Check my balance"
   • "Show my claims"
   • "What's the weather?"

Note: Works best in Chrome or Edge browser

🐛 TROUBLESHOOTING

Voice not working?
→ Use Chrome or Edge browser
→ Allow microphone permission
→ Speak clearly and wait

Weather not loading?
→ Check internet connection
→ API might be rate-limited (using fallback data)
→ Click refresh button

Location not working?
→ Allow location permission in browser
→ Check if location services enabled
→ Click "Update Location" button
